Tải test suite mẫu

Mình có chuẩn bị một file main.cpp có thể ghi kết quả output và copy các file input vào một thư mục. Các thư mục này có thể được zip lại và upload làm test suite.

  • Thay thế nội dung main.cpp bằng code trong file main-write.cpp.
  • Chỉnh lại các file input
  • Chạy chương trình, nhập số tt test N. Sau khi chạy, code sẽ tạo một folder có tên test"N"
  • Lặp lại các bước trên. Sau đó nén các thư mục test"N" vào một file zip. Upload file đó!

Ngoài ra các bạn có thể xem qua file main.cpp web app sử dụng để chấm bài.

(file này có output như file trên nhưng in ra màn hình thay vì viết vào file)
